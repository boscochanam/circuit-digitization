The pipeline repository is available at \url{https://github.com/boscochanam/circuit-digitization}. The component dataset source is the public CGHD1152 corpus \cite{Thoma2021}, together with the HDC recognition subset documented in the repository. The 23 image wire benchmark with 300 annotated wire segments is maintained by the authors. Benchmark outputs, figure assets, and manuscript source files are contained in the present project workspace.
